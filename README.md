# 📱 Programação de Aplicativos Mobile com Flutter

> Repositório do compoente Programação de Aplicativos Mobile I  - Ensino Médio com Habilitação Profissional de Técnico em Desenvolvimento de Sistemas Período Integral (DS-1/2026).

Este repositório tem como objetivo disponibilizar conteúdo para o componente de `Programação de Aplicativos Mobile I`, utilizando o framework Flutter. Durante o curso, os alunos aprenderão desde a configuração do ambiente até a criação de aplicativos completos com interfaces ricas e interativas.

# 🎯 Objetivos de Aprendizagem

* Compreender os conceitos fundamentais do Flutter e Dart
* Configurar o ambiente de desenvolvimento
* Criar interfaces responsivas com widgets
* Implementar lógica de navegação entre telas
* Consumir APIs e trabalhar com dados locais
* Publicar um aplicativo básico
