# Introdução a Linguagem Flutter

O Flutter é um conjunto de ferramentas de código aberto desenvolvida pela Google.
Lançado em 2018, oferecia suporte ao desenvolvimento de aplicações móveis. Hoje o Flutter oferece suporte ao desenvolvimento de aplicações em seis plaformas: `iOS`, `Android`, `Web`, `Windows`, `MacOs` e `Linux`.

##  Desenvolvimento Nativo x Multiplataforma
Flutter é uma estrutura de desenvolvimento multiplataforma.

### 📌 Desenvolvimento Nativo

No desenvolvimento nativo, a aplicação é criada especificamente para uma plataforma, como Android ou iOS.
Essa abordagem oferece:
* Acesso total aos recursos do dispositivo
* Alto desempenho e velocidade

Porém, exige:

* Mais código
* Mais tempo de desenvolvimento
* Equipes diferentes para cada plataforma
* Maior custo para manter aplicações multiplataforma

### 📌 Desenvolvimento Multiplataforma

No desenvolvimento multiplataforma, uma única base de código é usada para criar aplicações em várias plataformas.
Isso traz vantagens como:

* Redução de custos

* Desenvolvimento mais rápido

* Experiência de usuário mais consistente

O Flutter se destaca nesse modelo por oferecer desempenho próximo ao nativo e ferramentas eficientes para os desenvolvedores.

## 🚀 Vantagens do Flutter

* Alta performance: utiliza a linguagem Dart e compila diretamente para código de máquina, garantindo rapidez e eficiência.

* Renderização rápida e consistente: usa a biblioteca gráfica Skia, garantindo a mesma aparência em todas as plataformas.

* Facilidade de desenvolvimento: recursos como Hot Reload permitem visualizar alterações instantaneamente sem perder o estado da aplicação.

* Ferramentas avançadas: inspetor de widgets e suporte a bibliotecas da comunidade facilitam o desenvolvimento e a manutenção.

## 💻 Linguagem de Programação do Flutter

O Flutter utiliza a linguagem Dart, também desenvolvida pelo Google.
A Dart é otimizada para interfaces de usuário e possui recursos importantes, como:

* Segurança nula de som (Null Safety), que ajuda a evitar erros comuns e melhora a qualidade do código.

* Melhor produtividade e menos tempo gasto com correções.

## 🧩 O que são Widgets no Flutter?

No Flutter, tudo é widget.
Botões, textos, imagens, layouts e telas são construídos a partir de widgets.

Características dos widgets:

* São altamente personalizáveis

* Utilizam o conceito de composição, permitindo combinar widgets menores para criar outros mais complexos

* São renderizados pelo próprio mecanismo gráfico do Flutter, garantindo aparência uniforme em todas as plataformas

## 🧱 Tipos de Widgets no Flutter

O Flutter oferece um amplo catálogo de widgets organizados em categorias, como:

* Material (baseado no Material Design do Google)

* Cupertino (estilo iOS)

* Layout, estilo e animação

Além disso, fornece layouts e temas prontos, permitindo que o desenvolvedor comece rapidamente o desenvolvimento de aplicações.
